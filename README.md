# **Rational_enzyme_design**
Рациональный дизайн липазы B из Candida antarctica (CALB): структурно-функциональный анализ, докинг с p-нитрофенилбутиратом и предложение точечных мутаций для повышения активности.

# Задачи:
 1. Аннотировать ключевые функциональные остатки активного центра фермента CALB.
 2. Оценить качество PDB-структуры.
 3. Выполнить молекулярный докинг p-нитрофенилбутирата (pNPB) в активный центр CALB.
 4. Проанализировать ключевые взаимодействия фермент-субстрат.
 5. Предложить 3 точечные мутации с биохимическим обоснованием.
 6. Описать ограничения in silico-моделирования и экспериментальные риски.
# Контекст кейса:
Компания подбирает фермент-кандидат для прикладной реакции и рассматривает возможность его улучшения за счет точечных замен. Необходим предварительный in silico-анализ структуры, активного центра и взаимодействия с субстратом.
# Цель проекта:
На основе последовательности фермента CALB провести структурно-функциональный анализ, смоделировать связывание с субстратом и предложить 3 рациональные мутации для изменения свойств белка.
# Описание исходных данных: 
 1. Фермент Липаза В из Candida antarctica (CALB)
 2. Uniprot ID: P41365
 3. PDB ID: 1TCA (Разрешение 1.55 Å)
 4. Субстрат: П-нитрофенилбутират (pNPB)

# Описание этапов:
 *1. Сбор и подготовка данных:*
   Скачиваем PDB-структуру фермента CALB и генерируем 3D-структуру субстрата p-нитрофенилбутирата (pNPB).
 *2. Структурно-функциональный анализ:*
   В PyMOL находим и визуализируем ключевые остатки активного центра, оцениваем качество структуры.
 *3. Молекулярный докинг:*
   Сначала очищаем белок от воды, добавляем водороды и заряды, конвертируем в формат PDBQT, лиганд также конвертируем в PDBQT и задаём область поиска вокруг активного центра. Запускаем AutoDock Vina с конфигурационным файлом, получаем позы связывания, анализируем лучшую.
 *4. Дизайн мутаций:*
   На основе данных об активном центре и результатах докинга предлагаем 3 точечные мутации, для каждой указываем обоснование, ожидаемый эффект на активность/стабильность и возможные риски.
 ## *5. Оформление результатов:*
